# 🚀 QA-CICD: Selenium CI/CD Pipeline Demo

[![Java CI with Maven](https://github.com/NadunBandara/QA-CI-CD-Built/actions/workflows/test-pipeline.yml/badge.svg)](https://github.com/NadunBandara/QA-CI-CD-Built/actions/workflows/test-pipeline.yml)

A complete test automation framework demonstrating CI/CD integration with Selenium WebDriver, TestNG, and GitHub Actions.

## ✨ Features

- ✅ Page Object Model design pattern
- ✅ Cross-browser testing support
- ✅ GitHub Actions CI/CD pipeline
- ✅ Automated test reporting
- ✅ Parallel test execution
- ✅ Headless mode for CI environment
- ✅ Extent Reports integration

## 🛠️ Tech Stack

- Java 21
- Selenium WebDriver 4.27.0
- TestNG 7.8.0
- Maven
- GitHub Actions
- WebDriverManager
- Extent Reports

## 🚀 Quick Start

### Prerequisites
- Java 21 or higher
- Maven
- Git

### Clone and Run
```bash
git clone https://github.com/NadunBandara/QA-CI-CD-Built.git
cd QA-CI-CD-Built
